# 🚘 BMW Sales Intelligence Dashboard 

## Overview
This Power BI dashboard provides insights into BMW global sales performance from 2010–2024 using interactive visualizations and KPI analysis.

## Dataset
<a href= "https://github.com/Nimishaa6/bmw-sales-dashboard-powerbi/blob/main/BMW%20sales%20data%20(2010-2024)%20(1).csv">Dataset</a>

<a href= "https://github.com/Nimishaa6/bmw-sales-dashboard-powerbi/blob/main/Screenshot%202026-05-26%20103109.png">Dashboard</a>

## Dashboard Features
- Total Revenue Analysis
- Units Sold KPI
- Average Car Price
- Regional Sales Analysis
- Fuel Type Distribution
- Sales Trends
- BMW Model Performance

## Tools Used
- Power BI
- Data Visualization
- Data Analytics

## KPI Highlights
- Total Revenue: $457.32M
- Units Sold: 10.07K
- Avg Car Price: $75.03K
